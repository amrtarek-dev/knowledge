## dirb
DIRB (directory Buster) is a Web Content Scanner. 
It looks for existing (and/or hidden) Web Objects. 
It basically works by launching a dictionary based attack against a web server and analyzing the responses.
``` bash
dirb domain.com
# find domain.com/..
```
